# 🩺 FonoClinic Landing Page

Landing Page moderna e responsiva para uma clínica de fonoaudiologia, desenvolvida com foco em UX, acessibilidade e conversão.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

## 🚀 Tecnologias Utilizadas

- **HTML5 Semantic** para SEO e Acessibilidade.
- **Tailwind CSS** (via CDN) para estilização moderna e produtividade.
- **Google Fonts** (Outfit) para tipografia premium.
- **Phosphor Icons** para iconografia consistente.

## 📋 Funcionalidades

- [x] **Hero Section**: Com CTAs claros e design acolhedor.
- [x] **Serviços**: Cards detalhados sobre as especialidades.
- [x] **Sobre**: Seção de biografia profissional.
- [x] **Depoimentos**: Prova social de pacientes.
- [x] **Footer**: Informações de contato e redes sociais.
- [x] **Responsive Design**: Otimizado para Desktop, Tablet e Mobile.

## 🛠️ Como Executar Localmente

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente no seu navegador.
3. Ou utilize o Live Server (VS Code) para uma experiência de desenvolvimento em tempo real.
